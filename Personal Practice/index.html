Red-Black Tree

Definition

A rb-tree (red-black tree) is a self balancing binary tree, defined by a simple set of rules:

  1. All the nodes on a rb-tree are color Red or Black.
  2. All the leaves (NIL) or empty nodes are Black.
  3. The root of the tree is also Black.
  4. A Red node always has two rb-tree children.
  5. The black height hb is the number of black nodes in a path from the root of the tree to a leave. All the paths of the tree have the same hb .

This implementation ensures O(log n), where n is the number of elements, for the following operations: Search, Insert, and Delete.

Although we can create an rb-tree for any kind of comparable data, this implementation only works on Numbers.
